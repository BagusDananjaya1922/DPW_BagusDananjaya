Form Sederhana




Berdasarkan hasil pengujian form pada browser, tombol Reset berfungsi untuk menghapus atau mengembalikan nilai input ke nilai awal yang telah ditentukan pada atribut value.
Tombol Kirim (Submit) berfungsi untuk mengirim data yang telah diisi ke halaman tujuan sesuai dengan atribut action.
Penggunaan atribut target dengan nilai _self akan membuka hasil pengiriman pada tab yang sama, sedangkan _blank akan membuka hasil pengiriman pada tab baru.
Selain itu, penggunaan method="get" akan menampilkan data input pada URL sehingga mudah dilihat namun kurang aman, sedangkan method="post" tidak menampilkan data pada URL sehingga lebih aman untuk pengiriman data.
Dengan demikian, atribut target dan method sangat mempengaruhi cara data dikirim dan ditampilkan pada browser.
